# 선생님의 비서

영어공부방, 소형 영어학원, 1인 원장, 과외 선생님이 수업 후 학부모에게 보낼 안내 문구를 빠르게 만들 수 있도록 돕는 무료 웹 도구입니다.

## 프로젝트 소개

선생님들은 매일 보강 일정 안내, 공지 발송, 숙제 안내, 학습 진도 공유 같은 반복적인 커뮤니케이션 업무를 합니다. 이 앱은 그 문구를 처음부터 쓰지 않고 입력값만 채워 넣으면 바로 복사해 쓸 수 있는 형태로 만들어 주는 것을 목표로 합니다.

처음에는 외부 API 없이 템플릿 기반으로 동작하며, 추후 학생별 기록 저장과 AI API 연동까지 확장할 계획입니다.

## 현재 기능

- **보강계산기** — 반 이름, 시작일, 수업 요일, 목표 회차, 휴무일을 입력하면 실제 수업일·마지막 수업일·학부모 안내문을 자동 생성. 2026년 한국 공휴일 자동 반영, 캘린더 이미지 공유 지원.
- **공지비서** — 학원/공부방 공지 문구를 템플릿 기반으로 작성.
- **학부모 문자비서** — 학부모에게 보내는 개별 메시지(숙제, 결석, 칭찬 등) 템플릿.
- **월간 꿀팁** — 매월 영어 교실 운영 팁을 보여주는 콘텐츠 영역.

## 실행 방법

이 프로젝트는 Vite 기반 React SPA입니다.

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 개발 방향

1. **무료 도구로 진입 장벽 제거** — 가입·로그인 없이 바로 사용 가능.
2. **숙제톡을 핵심 유입 기능으로** — 매일 쓰는 기능을 가장 매끄럽게 만들어 재방문을 유도.
3. **템플릿 우선, AI는 나중** — 초기에는 정해진 템플릿으로 빠르게 가치 제공.
4. **단계적 SaaS 확장** — 학생 기록 저장 → 2주 리포트 → 발송 상태 관리 → 로그인/DB/AI API 연동 순서로 확장.
5. **모바일 우선 UI** — 선생님이 폰에서 바로 복사·발송할 수 있는 한 손 사용 흐름 유지.

## 기술 스택

`package.json` 기준 실제 사용 중인 스택입니다.

- **프레임워크**: React 18 (`react`, `react-dom` ^18.3.1)
- **번들러/개발 서버**: Vite ^5.4.10
- **스타일링**: Tailwind CSS ^3.4.17, PostCSS ^8.4.49, Autoprefixer ^10.4.20
- **유틸리티**: html-to-image ^1.11.13 (캘린더 이미지 내보내기)
- **분석**: Google Analytics, Microsoft Clarity (`src/utils/analytics.js`)
- **언어/런타임**: JavaScript (ESM, `"type": "module"`)
