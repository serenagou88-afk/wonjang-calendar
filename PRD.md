# PRD — 선생님의 비서

## 제품명

- 정식명: **선생님의 비서**
- 임시 확장명: **클래스로그 (ClassLog)** — 학생 기록·리포트 기능 확장 시 사용 예정

## 타깃 사용자

- 영어공부방 원장 (1인 운영)
- 소형 영어학원 원장·강사 (학생 수 ~50명 규모)
- 영어 1인 원장
- 영어 과외 선생님

공통 특성:

- 수업 외 운영 시간(공지, 문자, 일정 관리)을 최대한 줄이고 싶음
- 대부분 카카오톡으로 학부모와 소통
- 학생 관리 시스템(LMS)을 따로 두지 않거나 엑셀/메모장으로 관리
- 모바일에서 빠르게 작성·복사·발송하는 흐름이 중요

## 문제 정의

1. **반복 문구 작성 부담** — 숙제 안내, 보강 공지, 결석 안내, 학습 피드백 등 매일 비슷한 문구를 처음부터 작성.
2. **보강·휴무 계산 실수** — 공휴일, 학원 자체 휴무, 수업 요일을 직접 계산하다 회차 누락·오안내 발생.
3. **학부모 신뢰 유지 부담** — 학습 상황을 주기적으로 공유하고 싶지만 시간이 부족함.
4. **기존 도구의 과한 무게** — LMS·CRM은 기능이 많고 진입 장벽이 높음. 가볍게 쓸 수 있는 도구가 부족.

## 핵심 가치

- **바로 복사해 쓰는 결과물** — 모든 기능이 "복사하기"로 끝남.
- **0초 온보딩** — 가입·로그인 없이 첫 화면에서 바로 사용.
- **한국 영어 사교육 맥락에 맞춘 톤** — 정중한 / 친근한 / 짧은 카톡용 등 학부모 응대 톤 분리.
- **모바일 한 손 사용** — 폭이 좁은 카드형 UI, 큰 버튼.

## MVP 기능

현재 동작 중이며 리브랜딩 대상이 되는 4개 기능입니다.

1. **보강계산** (기존 보강계산기)
   - 반 이름, 시작일, 수업 요일, 목표 회차, 휴무일 입력 → 실제 수업일·마지막 수업일·총 회차 계산
   - 2026년 한국 공휴일 자동 반영
   - 정중한 / 친근한 / 짧은 카톡용 안내문 자동 생성
   - 월간 캘린더 이미지로 공유

2. **공지톡** (기존 공지비서)
   - 학원 공지(개강, 휴강, 시험, 행사 등) 문구 작성

3. **숙제톡** (기존 학부모 문자비서) — **핵심 유입 기능**
   - 학생 개별 또는 반 단위 숙제·학습 안내 문구 작성
   - 톤 선택, 키워드 기반 템플릿 채우기

4. **리포트** (기존 월간 꿀팁)
   - MVP에서는 월간 운영 팁 콘텐츠 노출
   - 차후 학생별 2주 학습 리포트로 진화

## 메뉴 구조

```
선생님의 비서
├─ 보강계산    (구 보강계산기)
├─ 공지톡      (구 공지비서)
├─ 숙제톡      (구 학부모 문자비서)  ← 핵심
└─ 리포트      (구 월간 꿀팁)
```

기본 진입 탭: **숙제톡** (리브랜딩 단계에서 변경 검토)

## 숙제톡 기능 상세

### 목적

수업 직후 학부모에게 보내는 "오늘 학습 내용 + 숙제 안내" 카톡을 30초 안에 작성.

### 입력 항목 (MVP)

- 학생 이름 (선택)
- 반 이름 (선택)
- 오늘 학습 내용 (자유 입력, 키워드 입력 가능)
- 숙제 내용 (자유 입력)
- 다음 수업일 (선택)
- 톤 (정중한 / 친근한 / 짧은 카톡용)

### 출력

- 카톡 발송용 완성된 문구
- 복사 버튼
- 다른 톤으로 다시 만들기 버튼

### 확장 (v2 이후)

- 학생별 저장된 반/이름 자동 입력
- 자주 쓰는 숙제 템플릿 즐겨찾기
- 학생별 발송 이력 기록
- 2주치 누적해 학부모용 리포트로 묶기

## 공지톡 기능 상세

### 목적

학원/공부방 전체 또는 특정 반에 보낼 공지 카톡 문구 작성.

### 입력 항목

- 공지 유형 (휴강 / 보강 / 개강 / 시험 / 행사 / 기타)
- 대상 (전체 / 특정 반)
- 핵심 내용 (자유 입력)
- 날짜·시간 (선택)
- 톤 (정중한 / 친근한 / 짧은 카톡용)

### 출력

- 완성된 공지 문구
- 복사 버튼

## 보강계산 기능 상세

기존 보강계산기 로직을 그대로 유지하되 명칭만 변경.

### 입력 항목

- 반 이름
- 수업 시작일
- 수업 요일 (월~일 다중 선택)
- 목표 회차
- 학원 자체 휴무일 (직접 추가)
- 안내문 톤

### 처리

- 2026년 한국 공휴일 자동 반영
- 수업 요일과 휴무일을 고려해 실제 수업일 계산
- 마지막 수업일과 휴무로 빠진 날 분리 표시

### 출력

- 총 수업 회차, 마지막 수업일
- 월간 캘린더 (수업/휴무/공휴일/마지막 배지)
- 학부모 안내문 (선택한 톤 기반)
- 캘린더 이미지 공유 카드

## 리포트 기능 상세

### MVP

- 월간 영어 교실 운영 꿀팁 콘텐츠 (현재 월간 꿀팁 메뉴 유지)
- 메뉴명만 "리포트"로 변경, 콘텐츠는 그대로

### v2 — 학생별 학습 리포트

- 숙제톡·공지톡 발송 이력을 학생별로 누적
- 2주 단위로 학습 내용·숙제 수행 여부·출결 요약
- 학부모에게 PDF/이미지/카톡 메시지 형태로 공유 가능

### v3 — AI 코멘트

- 누적된 학습 기록을 바탕으로 학생별 강점/약점 코멘트 자동 생성

## 추후 SaaS 확장 계획

| 단계 | 핵심 변화 | 비고 |
| --- | --- | --- |
| v1 (MVP) | 템플릿 기반 무료 도구, 로그인 없음, 로컬 상태만 | 현재 |
| v1.5 | 브라우저 로컬 스토리지에 학생/반 정보 저장 | 비용 0, 가입 없이 개인화 |
| v2 | 로그인 + 클라우드 DB, 학생별 발송 이력 | 유료화 시작 가능 |
| v2.5 | 2주 리포트 자동 생성, 카톡/이메일 발송 상태 관리 | 학부모 신뢰 확보 기능 |
| v3 | AI API 연동 (문구 생성, 학습 코멘트) | OpenAI/Anthropic 등 |
| v3+ | 학원 전체 운영 대시보드, 결제·수강료 관리 | 풀 SaaS |

## 제외 범위 (MVP 기준)

- 로그인·회원가입
- 서버 데이터베이스, 사용자별 데이터 저장
- 결제/수강료 관리
- AI 기반 문구 생성 (API 호출)
- 카카오톡 자동 발송 연동
- 출결/성적 자동 채점
- 한국어 외 다국어 UI
- 데스크톱 전용 레이아웃 (모바일 우선 유지)
- 2026년 외 연도의 공휴일 자동 반영 (수기 입력으로 대응)
